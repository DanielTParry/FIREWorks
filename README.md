# 🎆 FIREworks

A Python library for financial independence and retirement planning. FIREworks provides tools for calculating safe withdrawal rates and analyzing retirement withdrawal strategies.

## Features

- Safe withdrawal rate calculations
- Withdrawal strategy analysis
- Historical backtesting
- Portfolio sustainability modeling
